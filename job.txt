Core platform

Google Sheets = jobs + applications database
Google Docs = master resume, tailored resume, cover letter
Google Drive = folder structure + version history + file copies
Google Apps Script = orchestration and lightweight automation
Vertex AI / Gemini through Apps Script = scoring, insights, tailoring
Slack webhook = “application ready” notifications

Apps Script supports triggers, Google Workspace automation, and now has a Vertex AI advanced service for calling models from script. Google Docs supports atomic batchUpdate, and Drive supports copying files plus revision history, which is great for generating tailored resume docs from a master template while still keeping edit history.

Do 3 layers, not one giant LLM call:

Layer A — cheap deterministic scoring

Before calling any model, compute:

keyword overlap
required skill match
years-of-experience hints
title similarity
seniority match
location / visa / remote constraints
industry/domain match

You can do this with plain JS in Apps Script or a small Python service. No model needed.

Layer B — profile-aware NLP match

Build a compact profile object:

target roles
skill clusters
strongest experiences
industries
tools/tech
leadership level
geography / work authorization constraints

Then produce:

fit_score
must_have_missing
strengths_to_emphasize
risk_flags
resume_focus_labels

Layer C — model insights only for shortlisted jobs

Only call the model when deterministic score passes a threshold.

That will save a lot of tokens.

Because Apps Script now supports Vertex AI’s advanced service, you can keep the whole flow inside the Google ecosystem instead of bolting on another orchestration layer.




3) Resume tailoring and cover letter generation

Never rewrite from scratch. Always start from a master resume and apply constrained edits.

That means:

keep a master resume doc
copy it for each application
feed the model:
job description
your profile summary
allowed edit rules
sections allowed to change
sections not allowed to change
generate:
bullet-level edits
keyword insertions
summary tweak
optional cover letter

Then write those changes into a copied Google Doc, not the master.
